## Conventions

* Code blocks marked `python` are runnable against the version of Bulwark in
  the same repository — if a sample stops working, that is a bug.
* Filenames are linked relative to the repo root so they resolve both on
  GitHub and inside generated MkDocs sites.
* Diagrams are ASCII so they survive rendering on every viewer (Markdown
  preview, PDF export, terminal-based readers).
